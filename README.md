## [공공] 서울시 따릉이 자전거 이용 예측 AI모델 
> 공공 빅데이터인 서울시 따릉이 데이터를 이용하여 인공지능 모델 개발

## 각 데이터 예측 모델 버전 설명
- ex01.py 버전. 의사결정트리 회기 방식 사용
- ex02.py 버전. 랜덤 포레스트 회기 방식 사용, 데이터 학습 후 랜덤포레스트모델 예측변수의 중요도 함수 사용. 중요도 낮은 피쳐 제거.
